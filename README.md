# AHP
## 层次分析法

- Python2
- 和积法

#### 配置

默认使用9级评分，如果需要增加，修改`RI_dict`

```python
RI_dict = {1: 0, 2: 0, 3: 0.58, 4: 0.90, 5: 1.12, 6: 1.24, 7: 1.32, 8: 1.41, 9: 1.45}
```

修改`A_arr`改变矩阵，注意使用小数，否则计算结果为整数

```python
A_arr = [[1, 2, 1.0/3, 3],
        [1.0/2, 1, 1.0/3, 2],
        [3, 3, 1, 4],
        [1.0/3, 1.0/2, 1.0/4, 1]]
```

#### 执行

cd到`AHP`目录

```
python AHP.py 
```
